CSS External

Um CSS externo, como o próprio nome sugere, está fora do nosso documento! Analise aí o código fonte da página e veja o que foi feito na linha 8, dentro da área <head>.

As principais vantagens no uso desse método são:

  • Como o arquivo está externo, o seu conteúdo (HTML5) fica separado do estilo (CSS3). Isso deixa tudo mais organizado.
  • Um mesmo arquivo de estilo pode ser usado em várias páginas do meu site. Eu preparei uma 👉 segunda página aqui 👈 que usa o mesmo estilo.
